HTML5 : Structure, Texte & Liens
Maîtriser les balises essentielles, la structure d'un document HTML5 et la création d'hyperliens.
Exercice 1.1
Ma première page HTML5
HTML5
Facile
Objectif
Créer un document HTML5 valide avec la structure minimale correcte, en utilisant les balises de base pour présenter votre profil personnel.

Créez un fichier profil.html contenant une page de présentation personnelle respectant les contraintes suivantes :

  • Déclarer correctement le doctype HTML5 et l'encodage UTF-8
  • Définir la langue de la page en français (lang="fr")
  • Donner un titre significatif affiché dans l'onglet
  • Créer un titre <h1> avec votre nom, et un sous-titre <h2> avec votre filière
  • Rédiger un paragraphe de présentation d'au moins 3 phrases
  • Lister 5 compétences avec <ul>, et vos 3 matières préférées avec <ol>
  • Ajouter une ligne de séparation <hr> avant le pied de page
  • Terminer par un paragraphe avec votre email en <strong>
  • Bonus Ajouter des balises <mark>, <sub>, <sup> de manière pertinente
Rappel — Structure minimale

La page doit commencer par <!DOCTYPE html>, contenir les balises <html>, <head> (avec <meta charset> et <title>) et <body>.

Exercice 1.3
Tableau de notes d'étudiants
HTML5
Moyen
Objectif
Créer un tableau HTML sémantique complet avec fusion de cellules, en-têtes et pied de tableau.

Créez un fichier notes.html contenant un tableau des notes de 5 étudiants dans 4 matières.

  • Utiliser <thead>, <tbody> et <tfoot>
  • Les en-têtes doivent utiliser <th> avec scope="col" ou scope="row"

Créer des formulaires complets avec validation native, types d'input HTML5 et accessibilité.
Exercice 2.1
Formulaire d'inscription étudiant
HTML5 Formulaires
Moyen
Objectif
Maîtriser tous les types d'input HTML5, les attributs de validation et les bonnes pratiques d'accessibilité des formulaires.

Créez un fichier inscription.html contenant un formulaire d'inscription complet pour ISET Kasserine :

  • Champ Nom (type text) : required, minlength="3", maxlength="50"
  • Champ Prénom (type text) : mêmes contraintes
  • Champ Date de naissance (type date) : doit être après 01/01/1990
  • Champ Email (type email) : required
  • Champ Téléphone (type tel) : avec pattern="[0-9]{8}" et placeholder="12345678"
  • Menu Département (select) : au moins 5 options dont une option vide par défaut
  • Choix Niveau (radio buttons) : L1, L2, L3
  • Cases Langues (checkboxes) : Arabe, Français, Anglais, Autre
  • Champ Mot de passe : minlength="8", et un champ confirmation
  • Champ Photo (type file) : accepte uniquement JPG/PNG
  • Zone Observations (textarea) : 5 lignes
  • Tout champ doit avoir un <label> lié via for/id
  • Bonus Regrouper les champs avec <fieldset> et <legend> : "Informations personnelles", "Scolarité", "Sécurité"
Conseil accessibilité

Chaque <input> doit avoir un id unique correspondant au for de son <label>. Ajouter autocomplete sur les champs courants (name, email, tel…).

HTML5 Formulaires
Difficile
Objectif
Utiliser les types number et range, les attributs min/max/step, et afficher un résultat avec <output>.

  • Ajouter un champ type="range" pour le coefficient de chaque matière (1 à 4)
  • Utiliser <output> pour afficher la moyenne pondérée (calculée via l'attribut oninput)

Exercice 6.2
Carte avec effet flip 3D
CSS3 Animations
Difficile
Objectif
Réaliser un effet de retournement de carte en 3D au survol avec perspective, rotateY et backface-visibility.
  • Structure HTML : .carte-wrapper > .carte > .recto + .verso
  • Le .carte-wrapper : perspective: 1000px
  • La .carte : transform-style: preserve-3d; transition: transform 0.6s
  • Au hover sur .carte-wrapper : transform: rotateY(180deg)
  • Le .recto et le .verso : position: absolute; backface-visibility: hidden
  • Le .verso : transform: rotateY(180deg) par défaut
  • Le recto = présentation de cours, le verso = détails et bouton d'accès
  • Bonus Créer 3 cartes dans une grille flex avec des thèmes de couleur différents
CSS3 — Flip card structure
.carte-wrapper { width: 280px; height: 180px; perspective: 1000px; } .carte { width: 100%; height: 100%; position: relative; transform-style: preserve-3d; transition: transform 0.6s ease; } .carte-wrapper:hover .carte { transform: rotateY(180deg); } .recto, .verso { position: absolute; width: 100%; height: 100%; backface-visibility: hidden; border-radius: 12px; } .verso { transform: rotateY(180deg); background: #1a2744; }
